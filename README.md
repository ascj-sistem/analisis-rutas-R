# 📊 Análisis de rutas en R

Este proyecto analiza los tiempos de recorrido (en minutos) para tres rutas diferentes (A, B y C).

## 🎯 Objetivo
Determinar cuál ruta es más eficiente utilizando herramientas estadísticas en R.

## 🧪 Metodología
- Análisis exploratorio de datos
- Gráficos comparativos
- Prueba t de Student
- Estadística descriptiva

## 📁 Archivos
- `analisis.Rmd`: código y análisis reproducible
- `analisis.docx`: informe final en Word
-  `Apuntes`:      https://www.notion.so/2A1-Dilema-de-los-tres-caminos-O-dilema-dos-tr-s-caminhos-d9bf5ee82b1e8290b3ef816351a484c7?source=copy_link

## 📌 Resultados
- La ruta B es la más rápida
- Existe diferencia estadísticamente significativa entre A y B
- La ruta C es la más lenta y variable
